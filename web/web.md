# web

## 综述
该目录存放 Web UI 的静态资源。HTTP 服务对外提供页面、样式与脚本，用于查看智能体对话、组织结构，并提供配置与模块管理界面。

该目录包含 2 个文件与 2 个子目录，直接文件类型以 .html、.md 为主。

## 文件列表
- index.html: 功能：Web UI 主页面。责任：定义页面 DOM 结构与交互入口，并按依赖顺序加载样式与脚本。内部结构：包含左侧面板（智能体列表/总览/模块管理）、右侧对话区、多个弹窗（消息/智能体/岗位详情）、Toast 容器与工件管理器窗口；通过 <script> 按顺序加载 utils、API、组件与主应用脚本。
- web.md: 功能：本目录说明文档。责任：描述本目录的综述、文件列表与子目录列表。内部结构：包含“综述 / 文件列表 / 子目录列表”。

## 子目录列表
- css: 功能：样式资源目录。责任：提供 Web UI 的样式定义。内部结构：以 style.css 为主。
- js: 功能：脚本资源目录。责任：提供 API 封装、主应用逻辑、组件与通用工具函数。内部结构：包含 api.js、app.js，以及 components/、utils/ 子目录与目录说明文档。（详见 js/js.md）
